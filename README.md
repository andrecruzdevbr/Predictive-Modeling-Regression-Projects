# Projetos de Modelagem Preditiva com Regressão Linear Múltipla

Este repositório apresenta dois projetos de Ciência de Dados desenvolvidos utilizando Regressão Linear Múltipla para previsão de vendas e estimativa de preços de automóveis.

---

## Projeto 1 - Predição de Vendas

Este projeto aplica Regressão Linear Múltipla para prever as vendas com base nos investimentos em publicidade.

### Variáveis Utilizadas

- TV
- Rádio
- Jornal

### Técnicas Aplicadas

- Análise Descritiva
- Análise de Correlação
- Análise de Covariância
- Regressão Linear Múltipla
- Análise de Resíduos
- Análise de Multicolinearidade (VIF)
- Avaliação do Modelo

---

## Projeto 2 - Predição de Preço de Automóveis

Este projeto aplica Regressão Linear Múltipla para estimar o preço de automóveis utilizando características dos veículos.

### Técnicas Aplicadas

- Análise Descritiva
- Análise de Correlação
- Análise de Covariância
- Regressão Linear Múltipla
- Análise de Resíduos
- Análise de Multicolinearidade (VIF)
- Análise de Outliers
- Comparação de Modelos

### Modelos Desenvolvidos

- Modelo 1: Todas as Variáveis
- Modelo 2: 3 Variáveis Mais Correlacionadas
- Modelo 3: 5 Variáveis Mais Correlacionadas
